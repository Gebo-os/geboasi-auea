## What This Is

A complete, stunningly designed single-file full-stack web application for the **Advanced Unlabeled Expert Agent (AUEA)** — the fluid super-expert node in your GeboASI 20+ subsystem organism.

### Core Features

- **Lovable Sci-Fi UX/UI**: Premium cyber-neon dark theme (black + neon green/cyan), Tailwind + custom futuristic styling, smooth interactions
- **Dynamic Tone Switching**: 7 expert tones (Sovereign Strategist, Dark Truth Seer, Innovation Cortex, Tactical Executor, Symbio Companion, Cold Sun Protocol, Risk Defense) that adapt voice and emphasis while preserving structure
- **Mandatory 5-Part Structured Responses**:
  1. Core Insight (brutal truth + patterns)
  2. Strategic Routes (concrete options)
  3. Micro-Missions (7-day actionable sovereignty builders)
  4. Sovereignty & Risk Matrix
  5. Evolution & Activation
- **Powerful Local Simulation Engine**: Sophisticated client-side backend that generates high-quality structured responses with Gebo context awareness (STL sovereignty, builds, sigils, GeboASI, etc.)
- **Live Grok/xAI Integration**: One-click switch to real superintelligence. Your API key is stored only in browser localStorage. Full conversation history passed.
- **8 Expert Route Tools** (one-click activation):
  - 7-Day Micro-Mission Planner
  - Sovereignty Audit
  - Sigil Manifestation Layer
  - Cold Sun Protocol Sync
  - Multi-AI Orchestration
  - G-Droid Spec Generator
  - Biz Ops Professionalizer
  - Music Prod Workflow Optimizer
- **Persistent Chat + Sessions**: Auto-saves to localStorage, new sessions, export full history as JSON
- **Privacy-First**: 100% local by default. Nothing leaves your browser unless you explicitly use Live mode.
- **Regenerate, Copy, Metrics**: Full power-user controls

Open `index.html` directly in any modern browser and the expert node is instantly online.

## How to Use

1. **Start chatting** — Type anything. The node responds in perfect 5-part structure.
2. **Switch Tones** — Click any tone button in the left sidebar. The expert adapts instantly.
3. **Use Expert Tools** — Click any of the 8 tools to auto-fill powerful prompts.
4. **Go Live** — Click the LIVE GROK button, enter your xAI API key once. Now powered by real Grok.
5. **New Session** or **Export** — Keep context clean or archive your work.

**Pro tip**: Use the Local Simulation for quick private thinking. Switch to Live Grok when you need maximum depth or real-time research/creation.
